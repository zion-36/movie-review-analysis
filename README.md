# 映画レビューデータの傾向分析 (Movie Review Analysis)

本プロジェクトは、映画レビューデータを用いて、統計的な可視化とテキストマイニング（頻出語分析）を行うパイプラインです。

## 概要
研究活動で培ったデータ分析技術を公開可能な形で再構成したものです。
著作権保護のため、実データの代わりに独自に生成したシミュレーションデータ（dummy_reviews.csv）を使用しています。

## 分析内容
- **統計的分析**: 星評価の分布を可視化し、ユーザー満足度の傾向を把握。
- **時系列分析**: 投稿数の推移から、作品の話題性の変化を特定。
- **テキストマイニング**: 形態素解析を用いて、口コミ中の頻出単語を抽出。

## 使用技術
- Python 3.x
- Pandas, Matplotlib, Seaborn
- Janome (形態素解析)
- japanize-matplotlib (日本語表示対応)
